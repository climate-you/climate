# MapTiler Globe Starter

A minimal Vite project using the MapTiler JavaScript SDK to render an interactive 3D globe.

## Prerequisites

- Node.js 18+
- A MapTiler API key

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create your environment file:
   ```bash
   cp .env.example .env
   ```
3. Edit `.env` and set `VITE_MAPTILER_API_KEY`.
4. Start the dev server:
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
npm run preview
```
